== Description ==

T2S Store Locator - A simple and easy-to-use Multiple Store Locator plugin.

The Store Locator plugin for WordPress allows users to easily display a map of their store locations on their website. With the plugin, users can add and manage store locations, including details such as the store name, address, phone number, and hours of operation. When a user clicks on a store name, the plugin utilizes Google Maps to display the location on the map, making it easy for customers to find the nearest store. The plugin is a great tool for businesses with multiple locations, as it allows customers to easily find and visit their nearest store. Overall, the Store Locator plugin is a useful and convenient feature for any WordPress website that wants to display and manage multiple store locations.

The T2S Store Locator plugin for WordPress offers a range of features to help businesses manage their store locations more effectively:

* The plugin allows user to add, delete, and edit multiple stores information
* The plugin allows user to locate the store location by adding longitude, latitude, and address information
* The plugin allows user to access store locations by click store name, address, zip code, or other criteria.
* The plugin includes a search bar that allows visitors to search for store locations by name, address, zip code, or other criteria.
* The plugin is designed to be mobile-friendly, ensuring that customers can easily find store locations using their smartphones or tablets.

Overall, the Store Locator plugin is a powerful tool for businesses looking to manage their store locations more effectively and provide a convenient way for customers to find their nearest store.

== Installation ==

1. Install T2S Store Locator either via the WordPress.org plugin directory or by uploading the files to your server.
2. Go to menu: plugins –> Installed Plugins, activate T2S Store Locator plugin.
3. Create a [Google API Key](https://developers.google.com/maps/documentation/javascript/cloud-setup/).
4. Login into WordPress admin dashboard. Go to menu: Setting –> T2S Store Locator, add the Google API Key and setup the map settings and save.
5. Go to menu: Go to menu: Setting –> Permalinks, setup common settings and save.
6. Go to menu: Store –> Add Store, add your stores and save.
7. Copy and paste shortcode [T2S_StoreLocator] into any Page or Post and view page for map output.

That’s it. You’re ready to go!
